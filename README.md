**プロジェクト名称**
ぴたっとナース資産集

**プロジェクト概要**
現役看護師が「もう一人の家族」として高齢者の通院や生活を支援するサービス「ぴたっとナース」の事業ピッチ資料（Webスライド） を管理するプロジェクトです。

**主要な技術スタック**
プレゼンテーション: reveal.js (HTML/CSSベースのスライドフレームワーク)
フロントエンド: HTML5, CSS3 (CSS Variables, Flexbox/Grid), JavaScript (Vanilla JS)
データ集計・可視化: JavaScriptによる動的な会計ロジック、およびCSSを用いたカスタムチャート描画
デザイン: Google Fonts (Noto Sans JP), Material Icons

**ディレクトリ構造**
提供されたファイルに基づく、論理的な資産構造は以下の通りです。

Plaintext
famnomura/pitatto_nurse/
├── index.html            # サービス紹介用ピッチスライド（メイン）
├── img/                  # スライド用画像アセット（ロゴ、図解等）
└── mp3/                  # プレゼン演出用音声ファイル（着信音）

**ファイル一覧と解説**
主要なファイルとその責務について解説します。

index.html (事業紹介スライド)
役割: 投資家や提携先へ向けた事業ピッチ用インターフェース。
内容: 遠距離介護の課題（「120キロの壁」）、看護師による「医療通訳」の価値、収益モデル、3年間の事業計画などをアニメーションを交えて提示します。

img/ & mp3/ (マルチメディア)
役割: プレゼンテーションの視覚的・聴覚的リアリティの向上。
内容: 旭川から遠軽の距離を示す地図 や、重要な会議中の着信を演出する音声ファイル などが含まれます。

**エントリーポイント**
本プロジェクトは、目的に応じて2つの独立したWebページから動作を開始します。

事業説明（ピッチ）: index.html をブラウザで開きます。reveal.js が初期化され、キーボード操作によるスライド進行が可能になります。スライド5ページ目では、スマートフォンの着信音演出が自動再生されるインタラクティブな設計となっています。
